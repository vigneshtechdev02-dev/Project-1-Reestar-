# Reestar – Restaurant Landing Page

**Reestar** is a simple restaurant landing page developed using **HTML5** and **basic CSS**. This is a beginner-friendly frontend project created to practice webpage structure, styling, and layout design.

The website features a clean, single-page design with four food menu sections, each displaying an image, dish name, description, and an **Order** button. It also includes a **Contact Us** section where users can enter their name, age, and email. The project focuses on creating an attractive and user-friendly interface while improving fundamental HTML and CSS skills.

### Technologies Used

* HTML5
* CSS3 (Basic)

### Features

* Single-page responsive layout
* Four food menu cards
* Order button for each menu item
* Contact Us form
* Clean and simple user interface

This project helped me strengthen my understanding of HTML page structure, CSS styling, spacing, typography, and basic web design principles. It is one of my early frontend development projects and an important step in my journey toward becoming a Full-Stack Developer.
